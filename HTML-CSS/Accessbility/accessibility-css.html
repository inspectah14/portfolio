Tillgänglighet - vad innebär det?

Tillgänglighet innebär att man bygger platser och tjänster på så vis att utformningen av samhället är anpassat för att alla människor i allmänheten ska kunna ta del av informationen och tjänster. Detta oavsett om man har någon typ av funktionsvariation.

Tillgänglighet på webben betyder i precis samma sak fast på internet. Det vill säga att alla människor i samhället skall kunna ta del av information och tjänster som finns på webben, oavsett om de har någon form av funktionsnedsättning eller ej.

Aktuella Ämnen

Design

Presentation

Presentera innehållet i en meningsfull ordning för alla besökare av webbplatsen.

Responsivitet

Se till att allt innehåll presenteras rätt oavsett skärmens riktning. Skapa en flexibel layout som fungerar vid förstoring eller liten skärm.

Färgskillnader

Använd inte enbart färg för att förmedla information, komplettera betydelsebärande färgskillnader med någon annan synlig skillnad, exempelvis ramar eller fetstil.

Kontrast

Använd tillräckliga kontraster i komponenter och grafik. Använd tillräcklig kontrast mellan text och bakgrund.

Epilepsi

Orsaka inte epileptiska anfall genom blinkande. Maximalt tre växlingar från ljus till mörk eller tvärtom inom en sekund är acceptabelt.

Utveckling

Text i Bild

Använda CSS för att placera text i bilden, istället för att ha texten avbildad.

Fokus

Använd CSS för att tydligt markera vilket fält eller element som är i fokus.

Navigation

Utveckla systemet så att det går att hantera med enbart tangentbordet med exempelvis tab-knappen och piltangenterna. Erbjud möjlighet att hoppa förbi återkommande innehåll.

Semantik

Ange i kod vad sidans olika delar har för roll, använd semantiska taggar såsom h-taggar för rubriker och main-taggar för huvudinnehållet.

Språk

Ange huvudspråk med hjälp av lang-attribut på sidans rot-element.
