JavaScript 1-4 Courses

The frontend developer program began with the course JavaScript 1 during fall 2021 and once the program is finished, we will have completed the courses JS 1-4.

As of today, we have completed JS 1 and JS 2 and the below content is currently under construction and will continuously be updated.

Content

JavaScript 1 Course

Airport Check-In System

Project #1 for the JS1 course where we had to create an airport check-in system containing a regular queue button, a fast-track button, a check-in button, and a list containing the current passenger queue.
Quiz Project

Project #2 for the JS1 course. Some of the requirements included ten questions, radio buttons, checkboxes, a submit button, and a reset button. Additionally, once the submit button is pressed, the quiz is corrected and shows the user's score.

JavaScript 2 Course

Star Wars Trivia App

Individual project for the JS2 course, building a trivia app utilizing Star Wars API (SWAPI), fetch, async/await, and JS classes. Moreover, I organized my code and files in a structured manner, with the JS files divided into its different functionalities.
Webshop Group Project

Group project for the JS2 course, creating a webshop for a fictional company utilizing query strings and local storage, amongst other methods. Peers included Emma Olsson, Josefine Stübner, Albin Lindell, and Hanna Hansson Grapenros.
