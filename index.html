A short introduction of myself

Originally a marketer who sought a career change and ended up as a frontend development student, having been inspired by two close friends who are both frontend developers.

This website is a showcase of the projects I have undertaken throughout my time as a student at Nackademin. The portfolio not only portrays the skills I have learned but is also of accessible design. Please note that the website will be updated continuously as I progress through the program.

Lastly, you will find a short version of my resumé on its subpage. Here, you can also download the complete version as a PDF file.

HTML / CSS Course

Recipes Project

Project #1 for the HTML/CSS course where we were to create a web page containing three different recipes, with relatable images, external links, and fun facts.
Accessibility Project

Project #2 for the HTML/CSS course. The requirements included a fully accessible and responsive page built mobile-first and with semantic HTML tags.
Group Project

Project #3 for the HTML/CSS course. A group project with my classmates Vidar Rehn, Patrik Nyfeldt, and Mikael Roos, where we had to create a fully accessible and responsive site built mobile-first and with git collab.

JavaScript 1 Course

Airport Check-In System

Project #1 for the JS1 course where we had to create an airport check-in system containing a regular queue button, a fast-track button, a check-in button, and a list containing the current passenger queue.
Quiz Project

Project #2 for the JS1 course. Some of the requirements included ten questions, radio buttons, checkboxes, a submit button, and a reset button. Additionally, once the submit button is pressed, the quiz is corrected and shows the user's score.
